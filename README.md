# Azure Secure Infrastructure Lab

## Overview
This repository contains Infrastructure as Code (IaC) to deploy a secure, compliant Azure landing zone. It automates the creation of a Virtual Network, segmented Subnets, and Network Security Groups (NSG) to enforce Zero Trust principles.

## Technologies
* **Terraform**
* **Microsoft Azure**
* **Network Security Groups (NSG)**

## How to Use
1. Clone the repo.
2. Run `terraform init`.
3. Run `terraform apply`.
